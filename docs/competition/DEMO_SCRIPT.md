# CareerGo Demo Script (Competition)

Date: 2026-02-18  
Audience: Judges (web + education impact)

## 1) Opening (20-30s)

- Problem: học sinh khó chọn ngành vì thiếu dữ liệu cá nhân + thiếu trải nghiệm thực tế.
- Solution: CareerGo kết hợp `RIASEC + AI + VR + Community`.
- Goal: cá nhân hóa định hướng và tăng tự tin ra quyết định.

## 2) Student Journey (2-3 phút)

1. Mở `Test`:
- 50 câu RIASEC, tiến trình rõ ràng.
- Nêu nhanh autosave/chia phần để giảm mệt mỏi.

2. Mở `Results`:
- Hiển thị hồ sơ RIASEC + nhóm nghề `Ưu tiên`/`Dự phòng`.
- Click card nghề để mở VR preview.
- Chỉ vào block **Thảo luận cộng đồng gợi ý** (COM-201).

3. Mở `Chatbot`:
- AI đọc context học sinh (lớp, trường, mã RIASEC).
- Đặt 1 câu hỏi tư vấn ngắn.
- Chỉ vào block thảo luận cộng đồng liên quan để chứng minh kết nối hệ sinh thái.

## 3) Community Features (2-3 phút)

1. Tạo bài viết:
- Có tiêu đề + danh mục + nội dung.

2. Tương tác:
- Like bài viết.
- Thêm bình luận.
- Chủ bài đánh dấu "Phản hồi hữu ích" (COM-102).

3. Moderation:
- Báo cáo bài/bình luận với lý do (COM-103).

4. Trust + quality signals:
- Badge `Admin/Mentor` trên tác giả (COM-203).
- Bài ghim nổi bật (COM-104).
- Metrics widget cho tác động cộng đồng (COM-202).

## 4) Admin Mode (1-2 phút)

1. Vào `VR mode` bằng tài khoản admin:
- tải template + import nghề + phản hồi import rõ ràng.

2. Vào `Community`:
- pin/unpin bài viết,
- xem danh sách báo cáo cần kiểm duyệt.

## 5) Competition Closing (20-30s)

- Điểm mạnh khác biệt:
  - luồng end-to-end từ trắc nghiệm -> tư vấn -> trải nghiệm -> cộng đồng.
  - moderation + trust + metrics để đảm bảo tính bền vững.
- Sẵn sàng mở rộng:
  - COM-301 semantic retrieval.
  - COM-302 RAG assistant có trích dẫn.
